# 01. Setup and Forking Guide

## 1. Prerequisites
Before you begin, ensure you have the following installed:
- **Git**: [Download Git](https://git-scm.com/downloads)
- **Python 3.9+**: [Download Python](https://www.python.org/downloads/)
- **Visual Studio Code** (Recommended): [Download VS Code](https://code.visualstudio.com/)

## 2. GitHub Account
If you don't have one, create a [GitHub account](https://github.com/).

## 3. Forking the Repository
1. Navigate to the main repository page.
2. Click the **Fork** button in the top-right corner.
3. Select your personal account as the destination.

## 4. Cloning Your Fork
Open your terminal and run:
```bash
git clone https://github.com/<YOUR_USERNAME>/Automation_in_Everyday_Lab_Routine.git
cd Automation_in_Everyday_Lab_Routine
```

You are now ready to initialize your workspace. Proceed to **02_Submission_Guide.md**.
