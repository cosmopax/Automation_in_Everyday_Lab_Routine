# Repository Context for Jules / AI Agents

## Project Overview
This repository "Automation in Everyday Lab Routine" contains scripts (Python/Zsh) for laboratory automation, data handling, and student deliverables.

## Structure
- **main**: Core automation logic and scripts.
- **students_deliverables**: Storage for student protocols and achievements (25SS, 25WS).

## Coding Standards
- Python: PEP 8.
- Shell: Zsh (MacOS/Ubuntu compatible).
- Prefer robust, copy-paste ready CLI commands.
